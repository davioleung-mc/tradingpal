- 从模拟账户开始
- 学习风险管理原则
- 选择受监管的经纪商

### 需要了解的关键术语
- **杠杆**: 使用借入资本增加头寸规模
- **保证金**: 开设杠杆头寸所需的保证金
- **价差**: 买价和卖价之间的差异
- **滑点**: 预期价格和实际执行价格之间的差异
